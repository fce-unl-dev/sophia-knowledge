# Diplomaturas

Fichas de las diplomaturas universitarias superiores (nivel posgrado) y diplomaturas de pregrado de la FCE-UNL.

Cada archivo MD sigue el template canónico (`/template.md` en el raíz del repo).

## Naming

- Minúsculas, guiones medios, slug descriptivo. Sigla si existe: `gfsp.md`, `gto.md`.
- Para diplomaturas con nombre largo: usar un slug corto pero claro (`empresas-en-crisis.md`, `empresas-familiares.md`).
