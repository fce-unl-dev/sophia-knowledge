# Posgrados

Fichas de las carreras de posgrado de la FCE-UNL: **1 doctorado + 6 maestrías + 4 especializaciones**.

Cada archivo MD sigue el template canónico (`/template.md` en el raíz del repo).

## Naming

- Usar minúsculas y guiones medios. Sigla preferida si existe: `mba.md`, `maf.md`, `mcya.md`. Si no, slug descriptivo: `sindicatura-concursal.md`.
- Un archivo por programa, sin sufijos de fecha (la versión la lleva git).
