# Manual alumnos - carga de comprobantes de pago

Este documento describe el procedimiento paso a paso para cargar comprobantes de pago de inscripciones a posgrados en el sistema de la Facultad de Ciencias Económicas (FCE) de la UNL.

---

## Contenido

### Proceso de Carga de Comprobantes de Pago

Para cargar un comprobante de pago de una inscripción a posgrado, sigue los siguientes pasos:

1.  **Acceso al Sistema de Inscripciones de Posgrados**:
    *   Ingresa al sistema a través de la siguiente URL: [https://fce.unl.edu.ar/posgrados](https://fce.unl.edu.ar/posgrados)
    *   Utiliza el e-mail y la contraseña que elegiste al crear tu cuenta.
    *   **Importante**: Este sistema es independiente del SIU Guaraní. No comparten usuario ni contraseña.

2.  **Navegación a Inscripciones**:
    *   Una vez dentro del sistema, dirígete a la sección "Mis inscripciones".
    *   Selecciona la opción "carreras de posgrado".

3.  **Carga del Comprobante**:
    *   Busca y haz clic en la opción "Cargar comprobante de pago realizado".

4.  **Completar Datos del Comprobante**:
    *   El sistema te solicitará una serie de datos relacionados con el pago.
    *   Completa todos los campos requeridos.
    *   Presiona "Enviar" para grabar el formulario junto con el comprobante de pago.

5.  **Verificación del Comprobante**:
    *   Una vez grabado, el comprobante aparecerá listado debajo del formulario.
    *   Podrás visualizar el estado del pago en todo momento, indicando si está "pendiente" o si ya fue "facturado".
