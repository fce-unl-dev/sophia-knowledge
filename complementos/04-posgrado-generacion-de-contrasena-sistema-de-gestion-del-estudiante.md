# Generación de contraseña - Sistema de Gestión del Estudiante

Este documento describe el proceso para generar la contraseña inicial para acceder al Sistema de Gestión del Estudiante, dirigido a estudiantes de posgrado que han completado su inscripción.

---

## Contenido

### Proceso de Generación de Contraseña para Estudiantes de Posgrado

1.  **Recepción de Correo Electrónico Post-Inscripción:**
    *   Luego de realizar la inscripción, el estudiante recibe un correo electrónico.
    *   Este correo indica la necesidad de digitalizar los requisitos solicitados para la inscripción y proporciona el link de ingreso al Sistema de Gestión del Estudiante.

2.  **Acceso a la Función "Recuperar Contraseña":**
    *   Para poder acceder al sistema, se debe generar la contraseña desde la opción **"Recuperar Contraseña"**.

3.  **Ingreso de Datos para Recuperación:**
    *   Al ingresar a la acción "Recuperar Contraseña", se solicita el **DNI** del estudiante.
    *   También se solicita el **usuario**, que está conformado por las letras "dni" seguidas del número de documento (ejemplo: dni12342134).
    *   Una vez completados los datos, se debe presionar **"Siguiente"**.

4.  **Generación de Contraseña y Confirmación de Envío:**
    *   El sistema informará los datos declarados.
    *   Se debe presionar **"Generar Contraseña"**.
    *   Inmediatamente, el sistema muestra un cartel en la parte superior de la pantalla que indica que se envía un correo electrónico para generar la contraseña.

5.  **Recepción del Correo Electrónico para Resetear Contraseña:**
    *   Se recibirá un correo electrónico desde la dirección **gestiondelestudiante@servicios.unl.edu.ar**.
    *   Este correo contendrá las indicaciones para poder resetear la contraseña.

6.  **Establecimiento de Nueva Contraseña:**
    *   Presionando el link provisto en el correo electrónico, se redirige al sistema nuevamente.
    *   En esta pantalla, se deben completar los datos referidos al cambio de contraseña, ingresando la **nueva contraseña**.
    *   Una vez completada la información, se presiona el botón **"Guardar"**.

7.  **Confirmación de Contraseña Restablecida:**
    *   El sistema mostrará un mensaje de "cambio de contraseña aprobado" o "contraseña restablecida".

A partir de este momento, el estudiante puede ingresar al sistema con los datos de usuario y la contraseña generada.
