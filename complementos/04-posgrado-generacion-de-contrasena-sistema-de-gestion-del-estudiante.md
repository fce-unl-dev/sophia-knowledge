# Generación de contraseña - Sistema de Gestión del Estudiante

Este documento describe el proceso paso a paso para generar una contraseña inicial o restablecerla en el Sistema de Gestión del Estudiante de la FCE UNL, dirigido a estudiantes de posgrado.

---

## Contenido

### Proceso de Generación de Contraseña

#### 1. Recepción del Correo de Inscripción
Luego de realizar la inscripción, el estudiante recibe un correo electrónico que indica la necesidad de digitalizar los requisitos solicitados. Este correo incluye el enlace de ingreso al Sistema de Gestión del Estudiante.

#### 2. Acceso a la Función "Recuperar Contraseña"
Para poder acceder al sistema, se debe generar la contraseña desde la opción **"Recuperar Contraseña"**.

#### 3. Ingreso de Datos para Recuperación
Al ingresar a la acción "Recuperar Contraseña", se solicitará:
*   **DNI del estudiante**
*   **Usuario generado**: El usuario está conformado por las letras "dni" seguidas del número de documento (ejemplo: `dni12342134`).

#### 4. Confirmación y Envío de Correo para Generar Contraseña
1.  Complete los datos y presione **"Siguiente"**.
2.  El sistema informará los datos declarados. Presione **"Generar Contraseña"**.
3.  Inmediatamente, el sistema mostrará un mensaje indicando que se ha enviado un correo electrónico para generar la contraseña.

#### 5. Recepción del Correo de Restablecimiento
Recibirá un correo electrónico desde la dirección `gestiondelestudiante@servicios.unl.edu.ar` con las indicaciones para poder resetear la contraseña.

#### 6. Establecimiento de la Nueva Contraseña
1.  Presionando el link provisto en el correo, se redirigirá al sistema nuevamente.
2.  Complete los datos referidos al cambio de contraseña, ingresando la nueva contraseña deseada.
3.  Presione el botón **"Guardar"**.
4.  El sistema mostrará un mensaje de confirmación indicando que la contraseña ha sido restablecida.

#### 7. Acceso al Sistema
A partir de ese momento, podrá ingresar al sistema con los datos de usuario y la contraseña generada.
