# Cursos de Formación

Fichas de los cursos de formación profesional de la FCE-UNL (~22 cursos).

A diferencia de posgrados/diplomaturas, los cursos suelen tener planes de estudio más breves y un ciclo de oferta corto. Las fichas pueden ser **más concisas** que las de posgrado (alrededor de la mitad de longitud), pero siguen el mismo template para consistencia.

## Naming

- Minúsculas, guiones medios. Si tiene sigla, usarla: `trading-360.md`, `data-analytics.md`.
- Cursos con nombre largo: slug corto pero claro (`atencion-al-cliente.md`, `ingles-negocios.md`).
